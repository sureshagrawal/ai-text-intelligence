# AI-TEXT-INTELLIGENCE System

A production-ready AI system with clean separation of concerns.

## Architecture
- Frontend: Web / PWA / Mobile shells
- Backend: FastAPI (API, Auth, Business logic)
- ML: Independent ML system using MLflow (training + registry)

## Deployment (Independent)
- Frontend → Vercel
- Backend → Render / Railway
- MLflow → Render / VM / Fly.io

## Tech Stack
- Frontend: React / Next.js + Tailwind CSS
- Backend: FastAPI
- Machine Learning: MLflow + NLP models
